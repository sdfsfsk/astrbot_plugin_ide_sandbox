向已有文件末尾追加内容。

使用场景：
- `ide_write_file` 写入骨架后，用此工具逐段追加内容。
- 需要分多次写入同一个大文件，避免单次 content 过大导致 LLM 超时。

⚠️ 重要规则：
- 每次追加内容不要超过单次写入上限。
- 大型项目优先拆成多个模块文件，而不是把所有内容追加到单一文件。
- 目标文件必须先存在，不存在时请先用 ide_write_file 创建。

Args:
    filename(string): 文件名（不含路径），或超级管理员使用的绝对路径。
    content(string): 要追加到文件末尾的内容。

Returns:
    操作结果说明。
